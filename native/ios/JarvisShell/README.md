# Jarvis Shell

This folder is for the native iPhone 6 Jarvis shell. The final interface must be UIKit or Swift compatible with old iOS constraints. It must not be a web view shell.

Current state: skeleton notes only. No device behavior is claimed.

Initial screens:

- Home state
- Command screen
- Camera and inspection
- Sensor mode
- Capability browser
- Memory and logs
- Dock and sync
- Safety and permissions
