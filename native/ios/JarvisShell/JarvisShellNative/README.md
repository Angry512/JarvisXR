# JarvisShellNative

Native UIKit skeleton for the future Jarvis shell on iPhone 6. These files are Objective-C planning skeletons. They are not compiled proof and have not been deployed to an iPhone.

Target:

- iPhone 6 portrait-first layout.
- UIKit screens, not web views.
- Future bridge to `jarvisd` through `JVSCommandClient`.
- Offline identity first, with online and dock indicators only when available.
