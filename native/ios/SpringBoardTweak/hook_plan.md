# Hook Plan

Install empty tweak. Confirm SpringBoard stability.

## Stage 1

Log SpringBoard lifecycle events only. No behavior changes.

## Stage 2

Auto-launch Jarvis shell after unlock or respring if stable.

Experiment with home return behavior. Keep disable switch and uninstall package ready.

Investigate button or gesture activation. Do not assume access before testing.

## Evidence Required

Each hook needs device version, package version, log output, observed behavior, failure mode, and rollback confirmation.
