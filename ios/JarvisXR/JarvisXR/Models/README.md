# JARVIS Core ML Model Drop-In

Place a compiled Core ML model here when object detection is ready for device testing.

Accepted bundle names checked by the app:

- `JarvisObjectDetector.mlmodelc`
- `YOLO.mlmodelc`
- `YOLOv8n.mlmodelc`
- `ObjectDetector.mlmodelc`

The current app does not bundle a YOLO or object detection model. OCR and barcode scanning use Apple Vision after camera capture.
