# Setup Order

1. Build and install JARVIS with GitHub Actions and Sideloadly.
2. Launch JARVIS and test local commands.
3. Test deep links with Shortcuts Open URL actions.
4. Create Shortcuts from `shortcuts_recipes.md`.
5. Enable Voice Control and test built-in commands.
6. Add Vocal Shortcuts for JARVIS phrases.
7. Optional: configure AssistiveTouch shortcuts.
8. Optional: configure Switch Control recipes.
9. Test return path to JARVIS.
10. Test the Guided Access exit path.
11. Enable Guided Access only after every recovery path works.
