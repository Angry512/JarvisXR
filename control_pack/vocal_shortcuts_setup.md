# Vocal Shortcuts Setup

Vocal Shortcuts can trigger actions without making Hey Siri the JARVIS identity.

Suggested phrases:

| Phrase | Target action |
|---|---|
| Jarvis | Open JARVIS |
| Jarvis inspect | Run Shortcut: JARVIS Inspect |
| Jarvis quiet | Run Shortcut: JARVIS Quiet Mode |
| Jarvis normal | Run Shortcut: JARVIS Normal Mode |
| Jarvis diagnostics | Run Shortcut: JARVIS Diagnostics |
| Jarvis voice test | Run Shortcut: JARVIS Voice Test |
| Jarvis standby | Run Shortcut: JARVIS Standby |
| Jarvis online | Run Shortcut: JARVIS Online |

Setup outline:

1. Install JARVIS.
2. Create the Shortcuts listed in `shortcuts_recipes.md`.
3. Open Settings.
4. Open Accessibility.
5. Open Vocal Shortcuts.
6. Add each phrase and map it to the matching Shortcut.
7. Test each phrase with the phone unlocked.
8. Test again before using Guided Access.

Limit: Vocal Shortcuts trigger allowed actions. They do not grant root access or hidden system control.
