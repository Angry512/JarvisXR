# Shortcuts Recipes

Create these manually on the iPhone after JARVIS is installed.

## Open JARVIS

Actions:

1. Open App: JARVIS
2. Optional: Speak Text: JARVIS online

## JARVIS Inspect

Actions:

1. Open URL: `jarvis://inspect`

## JARVIS Quiet Mode

Actions:

1. Open URL: `jarvis://command?text=quiet%20mode`

## JARVIS Normal Mode

Actions:

1. Open URL: `jarvis://command?text=normal%20mode`

## JARVIS Diagnostics

Actions:

1. Open URL: `jarvis://diagnostics`

## JARVIS Search

Actions:

1. Ask for Input: Search query
2. Open Safari search URL or use a Search Web action if available.
3. Optional: Show Result: Return to JARVIS when finished.

This requires Wi-Fi for web search. Offline fallback is JARVIS local notes and tools.

## JARVIS Camera

Actions:

1. Open URL: `jarvis://inspect`

Alternative:

1. Open App: Camera

## JARVIS Return

Actions:

1. Open URL: `jarvis://command?text=status`

## JARVIS Voice Test

Actions:

1. Open URL: `jarvis://command?text=voice%20test`

## JARVIS Standby

Actions:

1. Open URL: `jarvis://standby`

## JARVIS Online

Actions:

1. Open URL: `jarvis://online`

## JARVIS Night Appearance Helper

Actions if available on the device:

1. Set Appearance: Dark
2. Set Brightness: desired value
3. Set Volume: desired value
4. Set Focus: chosen Focus mode
5. Open URL: `jarvis://command?text=status`

If an action is not available in Shortcuts on this device, skip it rather than using a fake substitute.
