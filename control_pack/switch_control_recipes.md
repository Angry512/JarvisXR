# Switch Control Recipes

Switch Control can replay repeated gestures, but it is more fragile than Voice Control.

Useful cases:

- Repeat a scroll gesture.
- Tap a known point in a stable app screen.
- Move through menus with scanning.
- Trigger an accessibility menu action.

Setup outline:

1. Open Settings.
2. Open Accessibility.
3. Open Switch Control.
4. Add a switch or use screen input.
5. Create a recipe for a repeated gesture.
6. Test with Guided Access off.
7. Use only for stable screens.

Do not rely on Switch Control for hidden app control, screen reading, or arbitrary automation. If the UI moves, coordinate taps can fail.
