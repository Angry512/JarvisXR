# JARVIS Control Pack

This folder is the manual non-jailbreak control layer for the iPhone XR.

JARVIS is still the visible brain and command surface. The Control Pack adds the official iOS layers that let the phone feel closer to a JARVIS device:

- Voice Control for global navigation, taps, scrolling, dictation, and app opening.
- Vocal Shortcuts for action phrases without relying on Hey Siri as the identity.
- Shortcuts for allowed multi-step actions.
- `jarvis://` deep links for sending commands into JARVIS.
- App Intents where the GitHub Actions Xcode build verifies support.
- Switch Control and AssistiveTouch recipes for optional repeated gestures.
- Guided Access after install if the device should stay in JARVIS.

Recommended command phrases:

- Jarvis inspect
- Jarvis return
- Jarvis mesh
- Jarvis quiet
- Jarvis normal
- Show Grid
- Tap the target number
- Scroll Down
- Go Home
- Take Screenshot

This does not claim jailbreak, root access, SpringBoard hooks, lock screen hooks, hidden screen reading, injected taps, or true OS takeover.
