# AssistiveTouch Recipes

AssistiveTouch can make common actions easier while keeping the device non-jailbroken.

Useful actions:

- Home.
- Screenshot.
- Control Center.
- Notification Center.
- Custom single or multi-finger gestures.
- Accessibility menu shortcuts.

Setup outline:

1. Open Settings.
2. Open Accessibility.
3. Open Touch.
4. Open AssistiveTouch.
5. Enable AssistiveTouch.
6. Configure custom actions or gestures.
7. Test every gesture before Guided Access.

Limits:

- It does not let JARVIS secretly control other apps.
- Coordinate gestures are fragile.
- It should support the appliance setup, not replace Voice Control.
