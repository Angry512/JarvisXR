# Vocal Shortcuts Setup For JARVIS

Vocal Shortcuts are the action phrase layer.

Recommended phrases:

- Jarvis
- Jarvis inspect
- Jarvis quiet
- Jarvis normal
- Jarvis diagnostics
- Jarvis voice test
- Jarvis standby
- Jarvis online

Map each phrase to a Shortcut from `control_pack/shortcuts_recipes.md`. Test each phrase before Guided Access.

Vocal Shortcuts do not replace JARVIS. They trigger allowed actions that return into JARVIS or use official iOS surfaces.
