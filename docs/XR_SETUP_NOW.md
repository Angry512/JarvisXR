# XR Setup Now

## A. Before Touching The Phone

1. Confirm the device is iPhone XR.
2. Confirm iOS is 18.7.9.
3. Do not update iOS.
4. Do not attempt random jailbreak tools.
5. Decide whether to erase/reset or keep existing data.
6. Note whether a Mac is available.
7. Note whether Apple Configurator is available.

## B. Strip The XR Down

1. Remove SIM if present.
2. Delete unnecessary apps.
3. Remove Home Screen clutter.
4. Disable unnecessary notifications.
5. Disable background distractions where iOS allows.
6. Configure Focus for Jarvis use.
7. Configure Screen Time restrictions.
8. Lock or hide unnecessary apps where iOS allows.
9. Leave only what supports Jarvis setup.

## C. Jarvis Identity Layer

1. Set the phone name toward Jarvis identity.
2. Use restrained, technical wallpaper direction.
3. Keep the app layout sparse.
4. Keep names and icons practical, not childish.
5. Make the lock screen visually align with Jarvis without claiming lock-screen control.
6. Make the phone feel like a dedicated instrument.

## D. Guided Access Immediate Route

1. Open Settings.
2. Enable Guided Access.
3. Set a Guided Access passcode or Face ID exit.
4. Open the Jarvis shell when available.
5. Triple-click the side button to lock into Jarvis.
6. Test exit before relying on it.

Guided Access can keep one app foreground. It cannot provide SpringBoard hooks, root access, lock screen hooks, launchd daemon install, or true OS ownership.

## E. Supervised Single App Mode Route

1. Confirm Mac access.
2. Confirm Apple Configurator or MDM-style setup access.
3. Confirm the XR can be supervised.
4. Confirm the Jarvis app build and bundle identifier exist.
5. Apply Single App Mode only after recovery steps are documented.

Supervised mode is stronger than Guided Access. It cannot be done from Windows alone. This repo provides a checklist, not fake commands.

## F. Restricted Jarvis Home Screen Fallback

Use this path if there is no Mac and no jailbreak:

1. Keep only essential apps.
2. Place Jarvis first when an app exists.
3. Use Focus and Screen Time.
4. Hide distractions.
5. Keep Spotify and search optional.
6. Treat the phone as a Jarvis-first appliance.

## G. Recovery Plan

1. Write down the Guided Access passcode.
2. Confirm Apple ID recovery access.
3. Confirm device passcode.
4. Test exiting Guided Access.
5. Test restart behavior before relying on lockdown.
6. Stop if you cannot prove the exit path.
