# Jarvis Appliance Mode Plan

v0.7 chooses this as the practical path for the only available hardware: iPhone XR on iOS 18.7.9.

Jarvis Appliance Mode is the best non-jailbreak plan. It does not claim true OS ownership. It makes the XR feel like a dedicated Jarvis device inside stock iOS limits.

## Setup Direction

- Erase or reset only if the user chooses.
- Keep no SIM.
- Remove unnecessary apps.
- Restrict app access.
- Configure the Home Screen so Jarvis is the visible purpose.
- Disable unnecessary notifications.
- Configure Focus.
- Configure Screen Time.
- Use supervised Single App Mode if Mac, Apple Configurator, or an MDM-style route is available.
- Use Guided Access as fallback.
- Make the native Jarvis shell the only visible experience.

## Jarvis Core

- Offline voice path.
- Camera and inspection tools.
- Sensor tools.
- Utility tools.
- Local memory.
- Local diagnostics.
- Local model adapters where realistic.

## What Is Secondary

Spotify and search are optional modules. They are not the identity. App orchestration is a secondary utility and should not become the main product.

## Recovery

Before locking down the device, document:

- Exit path.
- Recovery path.
- Backup path.
- How to remove restrictions.
- How to restore a normal phone state.

## Freeze Note

Do not add more app-control features before physical setup is attempted. The next evidence should be what the XR allows during setup.
