# XR Jailbreak Reality Check

This document was checked with current public references during v0.6. It is still a gate document, not a jailbreak instruction.

## Device Facts

- iPhone XR uses A12.
- iPhone XR is arm64e.
- iOS 18.7.9 is modern iOS.
- checkm8 and palera1n class routes do not apply to A12 iPhones.

## Current Gate

Trusted references checked during v0.6 indicate:

- iOS Guide lists iPhone XR as A12, arm64e, with latest jailbreakable version iOS 17.0.
- The Apple Wiki iOS 18.x jailbreak table lists iOS 18.7.9 for iPhones as no tool available.
- palera1n official materials describe support as A8 to A11 and Apple T2, not A12 iPhone XR.

Therefore true JarvisOS ownership on XR iOS 18.7.9 is blocked unless a new verified jailbreak appears.

## Verification Standard

Counts as verified:

- Official project page.
- Trusted jailbreak wiki.
- Reproducible GitHub project.
- Known developer release.
- Community validation from credible jailbreak channels.

Does not count:

- Random video.
- Paid unlocker site.
- Survey site.
- One click jailbreak claim.
- Site pretending palera1n supports unsupported devices.

## Safety Rule

Do not follow random YouTube jailbreak claims. Do not run paid unlocker tools. Do not attempt a jailbreak until the source, device, iOS version, exploit chain, recovery path, and community validation are clear.

## v0.7 Final Recommendation

Do not jailbreak this XR right now. If a credible jailbreak appears later, create a new feasibility pass before installing anything.
