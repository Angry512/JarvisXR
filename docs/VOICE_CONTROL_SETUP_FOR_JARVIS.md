# Voice Control Setup For JARVIS

Voice Control is the global navigation layer.

Setup:

1. Open Settings.
2. Open Accessibility.
3. Open Voice Control.
4. Enable Voice Control.
5. Test built-in commands: Show Grid, Tap, Scroll Down, Scroll Up, Go Home, Open Settings, Open Spotify, Open Safari, Dictate, Paste.
6. Add custom commands only after built-in commands work.

JARVIS command examples:

- Say Show Grid, then Tap a number.
- Say Scroll Down.
- Say Open Spotify.
- Say Open Safari.
- Say Go Home.
- Say Take Screenshot if supported.

Voice Control does not make JARVIS a root process. It controls visible iOS UI through Apple accessibility.
